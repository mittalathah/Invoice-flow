# üßæ InvoiceFlow

**InvoiceFlow** is a comprehensive, modern invoice management system built with React, designed for businesses to track sales, manage purchase invoices, and maintain client relationships with ease.

## üöÄ Key Features

- **Dynamic Dashboard**: Real-time financial insights with interactive charts showing revenue and expense trends.
- **Client Management**: Full CRM capabilities including detailed transaction histories and outstanding balance tracking.
- **WhatsApp Integration**: Send professional payment reminders directly via the WhatsApp Business Cloud API or deep-linking.
- **Multi-Role Access Control**: Built-in support for **Owners**, **Accountants**, and **Vendors** with specific permission sets.
- **Invoice Lifecycle**: Handles the entire flow from creation and multi-stage approval to payment recording.
- **Responsive Design**: Optimized for both desktop and mobile devices using Tailwind CSS.

## üì± WhatsApp Connection

The app supports two modes of WhatsApp connectivity:
1. **Direct Deep Link**: Uses `wa.me` for instant, free messaging via the user's WhatsApp client.
2. **Cloud API**: Pre-configured structure for Meta Graph API integration (requires API credentials).
